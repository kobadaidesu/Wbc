# ⚾ WBC 2023 Japan - 非公式ファンサイト

WBC（ワールド・ベースボール・クラシック）2023で優勝した侍ジャパンを記念した**非公式ファンサイト**です。

![WBC 2023](https://images.unsplash.com/photo-1464701116432-f476204d6c45?w=800)

## 🎯 プロジェクトの目的

このプロジェクトは以下の目的で作成されました：

1. **技術学習** - React、TypeScript、Tailwind CSSの学習
2. **UIデザイン** - スポーツサイトのデザインパターン習得
3. **ポートフォリオ** - Web開発スキルのデモンストレーション

## 🚀 技術スタック

- **Framework**: React 18
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4.0
- **Icons**: Lucide React
- **Build Tool**: Vite
- **Deployment**: (Vercel / Netlify / GitHub Pages 等)

## 📋 主な機能

### ✨ セクション一覧

1. **ヒーローセクション** - 優勝を祝う大型バナー
2. **ニュース** - 大会のハイライトニュース
3. **大会成績** - チーム統計とハイライト
4. **トーナメント表** - プール戦から決勝までの全対戦
5. **試合結果** - 全7試合の詳細スコア
6. **個人成績・表彰** - MVP、打撃・投手成績トップ5
7. **ハイライト動画** - 名場面集（UIのみ）
8. **タイムライン** - 大会期間中の出来事を時系列で
9. **選手名簿** - 優勝メンバー一覧
10. **歴代WBC** - 2006年から2023年までの歴史
11. **ギャラリー** - 画像ギャラリー
12. **フッター** - サイト情報とSNSリンク

### 🎨 デザイン特徴

- 日本代表カラー（紺 `#002868`、赤 `#BC002D`、金 `#FFD700`）を使用

## 🛠️ セットアップ

```bash
# リポジトリのクローン
git clone https://github.com/yourusername/wbc-2023-japan.git

# ディレクトリに移動
cd wbc-2023-japan

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## 📦 ビルド

```bash
# プロダクションビルド
npm run build

# ビルドのプレビュー
npm run preview
```
